# .github

Organization-level configuration for OhMyOpenSource!.

This repository powers two things:

- **`profile/README.md`**: the public profile page shown on the organization's GitHub page
- **Issue templates, PR template and Dependabot config**: applied automatically across all repositories in the organization

For contribution guidelines, standards and conventions, see [ohmyopensource-guidelines](https://github.com/ohmyopensource/ohmyopensource-guidelines).
