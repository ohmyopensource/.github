# OhMyOpenSource!

> Building free software, together.

OhMyOpenSource! is an open source organization dedicated to developing and sharing software that anyone can use, study, modify and distribute freely. Every project we build is open by default: the code, the process, and the standards.

- Read our [guidelines](https://github.com/ohmyopensource/ohmyopensource-guidelines) before contributing
